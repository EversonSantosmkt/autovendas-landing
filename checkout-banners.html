Como Salvar os Banners

  1. Use a Ferramenta de Captura do Windows (Win + Shift + S)
  2. Selecione apenas a área do banner desejado
  3. Salve a imagem como .PNG para manter a qualidade
  4. Faça upload no painel do seu checkout (Kiwify, Hotmart, etc.)

  5. Verde Sucesso: #00FF64
